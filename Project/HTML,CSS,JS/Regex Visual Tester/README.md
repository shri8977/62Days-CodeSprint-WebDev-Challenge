# 🔍 Regex Visual Tester

A beginner-friendly **regex testing tool** with **real-time match highlighting**, **group display**, and a simple **explanation panel** — built with HTML, CSS, and JavaScript.

---

## ✨ Features

- **Pattern input** with flags: global (g), case-insensitive (i), multiline (m)
- **Test string area** with live match highlighting
- **Match count** and **capturing groups** display
- **Beginner-friendly explanation** of pattern tokens and flags
- **Sample patterns** — email, phone, URL, digits, words
- **Error handling** for invalid regex syntax
- **Responsive** split-panel layout

---

## 🚀 How to Run

No installation needed.

1. Open `index.html` in any modern browser (Chrome, Firefox, Edge).
2. Enter a regex pattern or load a sample.
3. Type test text and watch matches highlight instantly.

### Optional — local server

```bash
npx serve .
```
