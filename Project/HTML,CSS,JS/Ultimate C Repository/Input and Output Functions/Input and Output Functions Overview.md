# Input and Output Functions in C

The `<stdio.h>` header provides Standard Input/Output functions that enable programs to communicate with users and devices.

**Key functions:**
- `printf()` — Formatted output to the console
- `scanf()` — Formatted input from the keyboard
- Format specifiers: `%d`, `%f`, `%c`, `%s`, `%lf`, etc.
- Escape sequences: `\n`, `\t`, `\"`, etc.
- Input validation and handling
