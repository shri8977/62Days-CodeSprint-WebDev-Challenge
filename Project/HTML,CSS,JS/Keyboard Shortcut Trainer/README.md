# ⌨️ Keyboard Shortcut Trainer

An interactive **keyboard shortcut practice game** for **VS Code** and **browser** shortcuts — with **score**, **accuracy**, and **streak** tracking. Built with HTML, CSS, and JavaScript.

---

## ✨ Features

- **Practice mode** for VS Code and browser shortcut categories
- **Random challenge generator** with action prompts
- **Real-time keyboard input** detection
- **Score, accuracy, and streak** tracking
- **Correct/incorrect feedback** after each round
- **Skip** option to reveal the answer
- **Responsive** dark-themed UI

---

## 🚀 How to Run

No installation needed.

1. Open `index.html` in any modern browser (Chrome, Firefox, Edge).
2. Choose a category and click **Start Practice**.
3. Click the practice area and press the shortcut shown.

### Optional — local server

```bash
npx serve .
```
