# Doctor Appointment Booking System

A full-stack MERN application that enables patients to book appointments with doctors based on specialization and available time slots.

## Features

- Patient Authentication (Login & Signup)
- Doctor Authentication
- Browse Doctors by Specialization
- Book Appointments
- Cancel Appointments
- Appointment History
- Role-Based Access Control (Patient / Doctor / Admin)
- Time Slot Conflict Prevention
- RESTful APIs
- MongoDB Database

## Tech Stack

### Frontend
- React.js
- React Router
- Axios
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcrypt
