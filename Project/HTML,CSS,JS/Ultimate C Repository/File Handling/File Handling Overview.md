# File Handling in C

File I/O in C uses functions from `<stdio.h>` to read from and write to external files.

**Key concepts:**
- Writing data to files using `fprintf()`
- Reading data from files using `fscanf()`
- Appending data to existing files
- Copying contents of one file to another
- Using `putw()` for writing integers
- File open modes: read (`"r"`), write (`"w"`), append (`"a"`)
