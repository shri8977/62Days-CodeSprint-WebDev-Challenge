### 📖 Overview

This is a self-contained, single-file, offline web application that provides a simulated Bash terminal environment. It allows you to practice common Unix commands in a safe sandbox right in your browser, without needing an internet connection or affecting your local file system. The state is ephemeral and resets on page reload.

---

### ✨ Features

*   **Offline & In-Browser**: Works entirely in your browser from a single HTML file. No server or installation required.
*   **Simulated Terminal**: An interactive terminal to execute commands.
*   **Virtual File System**: An in-memory file system to navigate and manipulate. It comes with a pre-defined directory structure.
*   **Core Command Support**: Implements several fundamental bash commands:
    *   `ls`: List directory contents.
    *   `cd`: Change directory.
    *   `pwd`: Print current directory.
    *   `mkdir`: Create a directory.
    *   `touch`: Create an empty file.
    *   `cat`: View and concatenate file contents.
    *   `echo`: Display text and append to files.
    *   `rm`: Remove files and directories.
    *   `clear`: Clear the terminal screen.
    *   `help`: Display available commands.
*   **Visual Desktop**: A graphical view that displays the files and folders in your current directory.
*   **Safe for Learning**: Experiment with commands without any risk to your actual computer.

### 🚀 How To Run

1.  **Clone the repo**

2.  **Open in Browser**:
    Open the downloaded HTML file in any modern web browser (like Chrome, Firefox, Safari, or Edge).

That's it! You can now start practicing your Bash commands.
